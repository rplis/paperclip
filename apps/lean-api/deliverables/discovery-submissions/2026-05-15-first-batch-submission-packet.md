100-character description:
Translate EPUB ebooks to new EPUB files while preserving chapters, images, links, and flow.

Long description:
EPUBtoEPUB is an EPUB-aware ebook translation tool. Upload an `.epub`, choose a target language, generate a free preview, then pay only if the sample looks right. The service parses the ebook in reading order, translates the text, and rebuilds a valid EPUB so readers keep chapters, navigation, images, metadata, and formatting cues instead of receiving loose translated text.

Best audience:
Language learners, personal readers, students, indie authors, translators, and small publishers who need a fast first-pass translated EPUB without manually unpacking, translating, and rebuilding the ebook.

Key features:
- EPUB in, translated EPUB out.
- Free preview before paying for the full translation.
- Preserves chapter order, reading flow, images, links, and metadata where possible.
- Upfront pricing based on book length.
- Stripe checkout for full translated file.

Suggested tags:
AI translation, EPUB translator, ebook translator, document translation, language learning, publishing, productivity, file conversion, localization
